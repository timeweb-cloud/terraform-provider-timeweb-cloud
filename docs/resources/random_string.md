# twc_random_string (Resource)

Resource that generates a random string. Lowercase letters (a-z) and digits (0-9) are always included; uppercase letters (A-Z) and special characters can be toggled via the upper and special arguments. The string is generated once on creation and stored in state. Changing symbol_count, upper or special forces a new resource to be created.


<!-- schema generated by tfplugindocs -->
## Schema

### Required

- `symbol_count` (Number) Number of characters in the generated random string

### Optional

- `special` (Boolean) Include special characters (!@#$%^&*()-_=+[]{}<>:?) in the generated string. Defaults to true.
- `upper` (Boolean) Include uppercase letters (A-Z) in the generated string. Defaults to true.

### Read-Only

- `id` (String) The ID of this resource.
- `result` (String) Generated random string
